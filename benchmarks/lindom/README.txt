This directory is used for benchmarking the Linear Domino Placement
game using the two different proof checker.

FILES:

Makefile

   Run "make run" to have it solve a set of benchmark files
   Run "make data" to have it tabulate the results
   Run "make clean" to delete unimportant files
   Run "make superclean" to have it delete the benchmark results

gen_lindom.py, writer.py
   Generate benchmark files.  
   Run "./gen_lindom.py -h" to see the list of commandline options

grab_data.py
   Used to tabulate result data
